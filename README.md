Install general dependencies: *"yarn install"*

Add veraPDF tool:
  3. (win) *"yarn veraPDF"*
  3. (linux) *"yarn veraPDF:linux"*
  3. (mac) *"yarn veraPDF:mac"*

Start app in the developer mode: *"yarn start"*

Production Build (produce the installers)
-  (win) : *"yarn build"*
-  (linux) : *"yarn build:linux"*
-  (mac) : *"yarn build:mac"*
